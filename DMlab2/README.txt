Точные вычисления комбинаторных чисел

Пакет программ предназначен для точного вычисления основных комбинаторных чисел, рассматриваемых в курсе
Основы дискретной математики для программистов. Пакет обеспечивает вычисления следующих
комбинаторных чисел:
1) Число размещений U(m,n)
2) Число размещений без повторений A(m,n)
3) Число перестановок P(n)
4) Число сочетаний C(m,n)
5) Число Стирлинга второго рода S(m,n)
6) Число Белла B(n)
******************************************************************************************************************************************

Структура проекта

libCalc.hpp - файл со стуктурой класса CalcComb, а так же с инициализацией используемых функций.
libCalc.cpp - файл с описанием функций (обьявленных в libCalc.hpp)
main.cpp - содержит исполняемую функция main()
******************************************************************************************************************************************

Функции

static unsigned int CalcU(unsigned int m, unsigned int n);
Функция для вычисления числа размещений
На вход: аргументы требуемые для вычисления n и m
На выходе: число размещений

static unsigned int CalcA(unsigned int m, unsigned int n);
Функция для вычисления числа размещений без повторений
На вход: аргументы n и m
На выходе: число размещений без повторений

static unsigned int CalcP(unsigned int m);
Функция для вычисления числа перестановок
На вход: аргумент n требуемый для вычисления
На выходе: число перестановок 

static unsigned int CalcC(unsigned int m, unsigned int n);
Функция для вычисления числа сочетаний
На вход: аргументы n и m, требуемые для вычисления
На выходе: число сочетаний

static unsigned int CalcS(unsigned int m, unsigned int n);
Функция для вычисления числа Стирлинга второго рода
На вход: аргументы n и m, требуемые для вычисления
На выходе: число Стирлинга второго рода

static unsigned int CalcB(unsigned int m);
Функция для вычисления числа Белла
На вход: аргумент n требуемый для вычисления
На выходе: число Белла

static void helpFunc(istream& istrm, ostream& ostrm)
Функция для вывода информации о доступных функциях

static unsigned int notZero(unsigned int arg1, unsigned int arg2)
Фунцикия возвращает произведение 2 аргументов. В случае переполнения, выводит сообщение об ошибке

static unsigned int Add(unsigned int arg1, unsigned int arg2)
Фунцикия возвращает сумму 2 аргументов. В случае переполнения, выводит сообщение об ошибке
